## 🐛 Troubleshooting

**Issue**: "Failed to connect to API"
```
Solution:
1. Verify service started: start-api.bat
2. Check port 5000 is available: netstat -ano | findstr :5000
3. Check CSV file exists: dir src/updated_data.csv
```

**Issue**: "Module not found: Flask"
```
Solution:
pip install -r requirements-api.txt
```

**Issue**: "CORS error in browser"
```
Solution: CORS is auto-enabled in policy_retriever_api.py
- No additional configuration needed
```

**Issue**: "Build fails with TypeScript errors"
```
Solution:
1. Run: npm install
2. Check Node version: node --version (should be 14+)
3. Rebuild: npm run build
```

See `API_DOCUMENTATION.md` Troubleshooting section for more.

## 📞 Integration Examples

### React Component - Search
```typescript
import { useState } from 'react';
import { PolicyRetrieverClient } from '@/api/policyRetrieverClient';

export function SchemeSearch() {
  const [results, setResults] = useState([]);
  const client = new PolicyRetrieverClient('http://localhost:5000');

  const handleSearch = async (query: string) => {
    try {
      const data = await client.search(query);
      setResults(data.results);
    } catch (error) {
      console.error('Search failed:', error);
    }
  };

  return (
    <div>
      <input onChange={(e) => handleSearch(e.target.value)} />
      <div>{results.map(s => <div key={s.id}>{s.scheme_name}</div>)}</div>
    </div>
  );
}
```

### React Component - Filter

const [filtered, setFiltered] = useState([]);

const handleFilter = async () => {
